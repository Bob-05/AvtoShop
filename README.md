# AvtoShop — Лендинг автосервиса и магазина автозапчастей

## 📋 Описание проекта

AvtoShop — это современный веб-сайт для автосервиса и магазина автозапчастей. Проект разрабатывается в рамках учебной практики командой из 5 специалистов.

Сайт предоставляет клиентам информацию об услугах, преимуществах компании, отзывах и контактных данных, а также включает административную панель для управления контентом.

## 🔐 Безопасность и хеширование паролей

Для защиты учетных данных администратора используется библиотека bcrypt.

### Принцип работы bcrypt

| Характеристика   | Описание                                      |
|------------------|-----------------------------------------------|
| Алгоритм         | Blowfish (адаптивная хеш-функция)             |
| Соль (salt)      | Генерируется автоматически и встраивается в хеш |
| Количество раундов | 10 (оптимальный баланс безопасности и производительности) |
| Необратимость    | Из хеша невозможно восстановить исходный пароль |

### Как это работает в проекте

- **При создании администратора** (прямая вставка в БД): Пароль `admin123` хешируется через bcrypt с 10 раундами соли. В базу данных сохраняется только хеш.
- **При входе в админ-панель**:
  1. Пользователь вводит пароль.
  2. Сервер берёт хеш из БД и сравнивает с введённым паролем через `bcrypt.compare()`.
  3. Если хеши совпадают — вход разрешён и генерируется JWT-токен.

**Пример кода (проверка пароля):**
```javascript
const bcrypt = require('bcrypt');

const inputPassword = req.body.password;
const hashFromDB = admin.password_hash;

const isValid = await bcrypt.compare(inputPassword, hashFromDB);

if (isValid) {
    // Генерация JWT и вход в систему
} else {
    // Ошибка: неверный пароль
}
```

## 🗄 Структура базы данных (MySQL)

База данных `avtoshop_db` состоит из трёх таблиц.

### Таблица `admin` — Учетные данные администраторов

| Поле           | Тип              | Ограничения                  | Описание                          |
|----------------|------------------|------------------------------|-----------------------------------|
| id             | INT              | PRIMARY KEY, AUTO_INCREMENT  | Уникальный идентификатор          |
| email          | VARCHAR(255)     | NOT NULL, UNIQUE             | Email для входа                   |
| password_hash  | VARCHAR(255)     | NOT NULL                     | Хеш пароля (bcrypt, 10 раундов)   |
| role           | ENUM('admin')    | DEFAULT 'admin'              | Роль пользователя                 |
| created_at     | TIMESTAMP        | DEFAULT CURRENT_TIMESTAMP    | Дата создания                     |

### Таблица `services` — Услуги автосервиса

| Поле         | Тип              | Ограничения                  | Описание                          |
|--------------|------------------|------------------------------|-----------------------------------|
| id           | INT              | PRIMARY KEY, AUTO_INCREMENT  | Уникальный идентификатор          |
| name         | VARCHAR(255)     | NOT NULL                     | Название услуги                   |
| description  | TEXT             | NOT NULL                     | Подробное описание                |
| price        | VARCHAR(100)     | NOT NULL                     | Стоимость                         |
| icon_url     | VARCHAR(512)     | —                            | Путь к иконке                     |
| created_at   | TIMESTAMP        | DEFAULT CURRENT_TIMESTAMP    | Дата создания                     |
| updated_at   | TIMESTAMP        | ON UPDATE CURRENT_TIMESTAMP  | Дата обновления                   |

### Таблица `reviews` — Отзывы клиентов

| Поле         | Тип              | Ограничения                  | Описание                          |
|--------------|------------------|------------------------------|-----------------------------------|
| id           | INT              | PRIMARY KEY, AUTO_INCREMENT  | Уникальный идентификатор          |
| name         | VARCHAR(255)     | NOT NULL                     | Имя автора                        |
| text         | TEXT             | NOT NULL                     | Текст отзыва                      |
| rating       | TINYINT          | NOT NULL, CHECK (1-5)        | Оценка                            |
| created_at   | TIMESTAMP        | DEFAULT CURRENT_TIMESTAMP    | Дата создания                     |

## 🚀 Запуск проекта

1. **Клонирование репозитория**
   ```bash
   git clone <url-репозитория>
   cd avtoservice-landing
   ```

2. Запуск фронтенда
Откройте файл index.html в браузере или используйте Live Server в VS Code:
 - Установите расширение Live Server
 - Нажмите правой кнопкой на index.html → Open with Live Server
 - Сайт откроется по адресу: http://127.0.0.1:5500

3. **Настройка и запуск бэкенда**
   ```bash
   cd server
   npm install
   npm start
   ```
   Сервер запустится на порту 5000.

4. **Настройка базы данных**
   Убедитесь, что MySQL Server установлен и запущен:
   ```bash
   net start MySQL80
   ```
   Подключитесь к MySQL и выполните SQL-скрипт:
   ```bash
   mysql -u root -p < database/create_db.sql
   ```
   Тестовые данные загрузятся автоматически.
